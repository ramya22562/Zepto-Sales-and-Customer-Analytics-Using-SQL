# Zepto Sales and Customer Analytics Using SQL

## Project Overview
This project analyzes Zepto sales and customer data using SQL to identify business trends, customer behavior, and sales performance. Various SQL queries were used to extract meaningful insights from the dataset and support data-driven decision-making.

## Tools Used
- SQL
- MySQL

## Key Analysis
- Total Sales Analysis
- Top-Selling Products
- Customer Purchase Behavior
- Revenue by Category
- Monthly Sales Trends
- High-Value Customers
- Order Performance Analysis

## Key Insights
- Identified top-performing products and categories.
- Analyzed customer purchasing patterns.
- Tracked sales growth and revenue trends.
- Evaluated customer contribution to overall sales.

## Files Included
- Zepto_Sales_Dataset.csv
- Zepto_SQL_Queries.sql
- Project_Report.pdf

## Skills Demonstrated
SQL Queries, Joins, Aggregations, Group By, Subqueries, Window Functions, Data Analysis, and Business Insights
